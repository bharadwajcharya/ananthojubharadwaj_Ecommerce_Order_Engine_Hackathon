# 🛒 E-Commerce Backend System (Django)

## 📌 Project Overview
This is a mini e-commerce backend system built using **Django** with server-side rendering (Templates + Static files).

The system simulates real-world e-commerce functionalities like product management, cart system, order processing, payment simulation, and stock handling.

---

## 🚀 Features

### 🧾 Product Management
- Add new products
- View all products
- Prevent duplicate entries
- Manage stock

---

### 🛒 Cart System
- Add items to cart
- Remove items from cart
- View cart items
- Calculate total price

---

### 📦 Order System
- Place orders from cart
- Convert cart → order
- Store order details

---

### 💳 Payment Simulation
- Random success/failure
- Handles real-world scenarios

---

### 🔁 Rollback Mechanism
- If payment fails:
  - Restore stock
  - Mark order as FAILED

---

### ❌ Cancel Order
- Cancel placed orders
- Automatically restore stock

---

### 📊 Order Management
- View all orders
- Track order status:
  - CREATED
  - PAID
  - FAILED
  - CANCELLED

---

### ⚠️ Inventory System
- Prevent overselling
- Show low stock products

---

## 🏗️ Tech Stack

- Backend: Django
- Frontend: HTML, CSS (Templates)
- Database: SQLite
- Language: Python
